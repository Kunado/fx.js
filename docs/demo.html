f(x).js

simple graph drawing for Web.

what is f(x).js?

f(x).js provides a simple way to draw a graph of a function on the Web.

MathJax is known as a way to display mathematical expressions on the web. This project aims to provide a tool like MathJax for graph drawing.

features

  • using web components
  • svg output

usage

  1. Load f(x).js.
  2. Put HTML tag <graph-fx></graph-fx> where you want to display the graph.
  3. Set graph options through <graph-fx> tag's attribute.
    Like this:
    <graph-fx
    function="x**2"
    range=5
    accuracy=.01
    width="180px"
    height="180px"
    ></graph-fx>
